实验2 MATLAB的数值运算

实验目的：了解MATLAB中矩阵和多项式的创建，以及数值函数的建立，掌握矩阵运算、多项式运算、数值函数的运算.
实验内容：首先，介绍TLAB中矩阵的创建、矩阵的运算;其次，介绍多项式的创建，多项式的四则运算，多项式的求导、求根、求值等运算; 最后，介绍数值函数的建立数值函数求值、求零点、求最大最小值、求定积分，数值函数作图等.

例2-31 分别用直角坐标和极坐标计算二重积分

% (1)直角坐标系下
fun = @(x,y) 1./( sqrt(x + y) .* (1 + x + y).^2 );
ymax = @(x) 1- x; %建立变上限函数
q = integral2(fun,0,1,0,ymax) %求被积函数在区域D上的数值二重积分
q = 0.2854
 
% (2)极坐标系下
fun = @(x,y) 1./( sqrt(x + y) .* (1 + x + y).^2 );
polarfun = @(theta,r) fun(r.*cos(theta),r.*sin(theta)).*r;
rmax = @(theta) 1./(sin(theta) + cos(theta));
q = integral2(polarfun,0,pi/2,0,rmax)
q = 0.2854
