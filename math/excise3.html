    $$ \begin{aligned} f&=\left(x_1^2-4 x_1 x_2+2 x_1 x_3\right)+x_2^2+2 x_2 x_3-2 x_3^2\\ & =\left(\left(x_1-2 x_2+x_3\right)^2-4 x_2^2-x_3^2+4 x_2 x_3\right)+x_2^2+2 x_2 x_3-2 x_3^2 \\ & =\left(x_1-2 x_2+x_3\right)^2-3 x_2^2+6 x_2 x_3-3 x_3^2 \\ & =\left(x_1-2 x_2+x_3\right)^2-3\left(x_2-x_3\right)^2 \\ & \text { 令 }\left\{\begin{array} { l r } y _ { 1 } = & x _ { 1 } - 2 x _ { 2 } + x _ { 3 } \\ y _ { 2 } = & x _ { 2 } - x _ { 3 } \\ y _ { 3 } = & x _ { 3 } \end{array} \text { 即 } \left\{\begin{array}{lr} x_1= & y_1+2 y_2+y_3 \\ x_2= & y_2+y_3 \\ x_3= & y_3 \end{array}\right.\right. \end{aligned} $$ 令$X=\boldsymbol{C}Y=\left( \begin{array}{ccc} 1 & 1 & 1\\ 0 & 1 & 1\\ 0 & 0 & 1 \end{array} \right)Y$. 则 $f$ 可化为标准形 $f=y_1^2-3 y_2^2$.
